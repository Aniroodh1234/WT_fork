Every HTML element is like a box with four layers
The box model is fundamental to CSS layout. From inside out: Content (your text/images), Padding (space around content), Border (visible outline), and Margin (space between elements).
Think of it like a picture frame: content is the photo, padding is the mat, border is the frame, and margin is the space between frames on the wall!
Total width = content + padding + border + margin
